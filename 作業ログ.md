- **GitHub**: https://github.com/belieeve/kintore-video-site（public）
- **ホスティング**: Vercel（無料プラン）
- **作業日**: 2026-04-12

## 実装した機能

### サイト機能
- YouTube風ダークテーマUI
- ヒーローバナー（ロゴ・概要・開催スケジュール表示）
- カテゴリ別セクション（Netflix風横スクロール）
- カルーセル矢印ボタン（◀ ▶）
- マウスドラッグ＆ホイール横スクロール（PC）
- カテゴリフィルター（ラジオ体操/筋トレ/有酸素/ストレッチ/ボクササイズ/ヨガ/ピラティス/ダンス/HIIT）
- 目的別フィルター（ダイエット/筋力アップ/柔軟性/リラックス/朝の運動/夕方の運動）
- 時間別フィルター（5分以内/5〜10分/10〜20分/20分以上）
- 部位別フィルター（足/腹筋/背中/胸/腕/お尻/体幹/肩/全身/上半身/下半身）
- 検索機能（YouTubeタイトル・カテゴリ・部位で検索）
- クリックでYouTubeに飛ぶ
- ホームに戻るボタン（フィルター表示中）
- 削除済み動画の自動非表示

### メニュー提案BOT
- 4ステップの質問（目的/時間/部位/レベル）で今日のメニューを自動生成
- 目的に応じた動画の組み合わせを提案
- 各動画クリックでYouTubeに飛ぶ

### 筋トレ記録表バナー
- 「筋トレしたら記録表に入力しよう！」バナー
- クリックでGoogleスプレッドシートの一番左のシート（最新月）に飛ぶ
- gid=0指定で月が変わっても自動対応

### 自動同期（GitHub Actions）
- 毎週月曜AM6:00(JST)に自動実行
- Google Sheets gviz APIでスプレッドシートからデータ取得（認証不要）
- YouTube oEmbed APIで動画タイトルを取得
- タイトルからカテゴリ・部位を自動分類
- 変更があればcommit & push → Vercel自動デプロイ
- 手動実行も可能（GitHub Actions → Run workflow）

### レスポンシブ対応
- PC / タブレット / スマホ対応
- スマホ: ハンバーガーメニュー、カテゴリチップス横スクロール
- ピンチズーム無効化
- 文字サイズ・カードサイズの自動調整（380px〜）

## セキュリティ対策
- XSS対策（動画タイトルのHTMLエスケープ）
- .gitignoreでxlsx・スクショ等を除外

## 今後のメンテナンス
- **スプレッドシート更新**: 毎週自動で反映される
- **新しい月のシート追加**: 「◯月メニュー」の命名規則に従えば自動で読み込まれる
- **記録表リンク**: gid=0なので月が変わっても対応不要
- **手動でデータ更新したい場合**: GitHubのActionsタブから手動実行
